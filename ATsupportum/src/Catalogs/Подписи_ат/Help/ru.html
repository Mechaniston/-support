Справочник «Подписи»

Для документа «Электронное письмо» реализована возможность подстановки в письмо подписи. Для каждого пользователя можно задать несколько подписей в справочнике «Подписи». В карточке пользователя на закладке «Подписи» можно задать разные подписи для разных учётных записей. Пользователь может просматривать и редактировать только свои подписи. Если на закладке «Подписи» в карточке пользователя нет настроенных подписей для данной учётной записи, то подставляется подпись, указанная для данной учётной записи в справочнике «Учётные записи электронной почты». Подпись можно подставить в письмо не только автоматически, но и при помощи контекстного меню области содержания письма (закладка «Письмо») или горячих клавиш Ctrl+Alt+P.
